Tugas No 1

Websit adalah sekumpulan halaman web yang bisa diakses melalui internet. Ketika seseorang memasukkan alamat web (URL) ke dalam peramban (browser), permintaan itu dikirim ke server web untuk mengambil halaman yang sesuai. Inilah beberapa komponen utama yang terlibat: Web Server: Ini adalah perangkat lunak atau perangkat keras yang menyimpan halaman web, berkas, dan konten lainnya. Ketika seseorang mengakses situs web, permintaan dikirim ke server web, yang kemudian mengirimkan halaman yang diminta kembali ke peramban pengguna. Domain: Domain adalah alamat yang digunakan untuk mengakses situs web. Ini adalah alamat yang mudah diingat yang mewakili alamat IP fisik server tempat situs web di-host. Contohnya adalah www.nama-situs.com. Domain membuat situs web lebih mudah diakses daripada mengingat alamat IP yang panjang. Hosting: Ini adalah layanan yang menyediakan ruang server untuk menyimpan konten dan data situs web agar dapat diakses secara online. Hosting memungkinkan situs web Anda dapat diakses oleh pengguna secara online. Ada berbagai jenis hosting seperti shared hosting, VPS hosting, dan dedicated hosting, yang menawarkan tingkat akses dan kontrol yang berbeda sesuai kebutuhan. Hubungan antara ketiganya dapat dijelaskan sebagai berikut: Situs web (konten, gambar, video, dll.) disimpan di server web. Server web ini memiliki alamat IP, tetapi karena alamat IP tidaklah mudah diingat, digunakanlah domain sebagai pengganti alamat IP. Ketika seseorang memasukkan nama domain di peramban, permintaan akan dikirimkan ke server DNS (Domain Name System) yang akan mengarahkannya ke alamat IP server web yang sesuai. Kemudian, server web ini akan mengeksekusi permintaan tersebut dan mengirimkan halaman yang diminta kembali ke peramban pengguna. Jadi, secara ringkas, website bekerja dengan server web menyimpan konten, domain memberikan alamat yang mudah diingat untuk mengakses konten tersebut, dan hosting memberikan ruang dan akses agar konten situs web dapat diakses secara online. Semuanya bekerja bersama untuk memastikan bahwa pengguna dapat mengakses dan menikmati situs web dengan mudah.
